# E-commerce-Angular
e-commerce app with angular & api
<br>This project was generated with <a href="https://github.com/angular/angular-cli">Angular CLI</a>.
<br>build using Angular material.
# Dependencies
- Angular CLI
- Register & Login & Logout
- Validation Form
- Authentication with JWT
- Routing
- Routing Guard
- JWT
- RXJS
- Respobsive Web Design
- Spinner
# Features
- Add product to cart
- Delete (All) product from cart
- Can't add (0) product to cart
- Can't add the same product to cart
- Make Order
- Show Product Details
- POP UP Idea
- Filtter products by category
# Technologies

- Jquery
- Javascript
